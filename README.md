# Geometric Representations in Robotics — IROS 2026 Workshop Website

This repository hosts the website for the **2nd Workshop on Geometric-Aware Representations: Structured priors, metrics, constraints, and groundings for Manipulation** at IEEE/RSJ IROS 2026.

## Workshop Details

- **Conference:** IEEE/RSJ International Conference on Intelligent Robots and Systems (IROS 2026)
- **Location:** Abu Dhabi, UAE
- **Date:** TBD, October 2026

## Website Structure

| File | Description |
|------|-------------|
| `index.html` | Home page |
| `speakers.html` | Invited Speakers |
| `organizers.html` | Organizing Committee |
| `program.html` | Workshop Program |
| `cfp.html` | Call for Papers |
| `submission.html` | Paper Submission |
| `404.html` | 404 Error page |
| `public/css/` | CSS stylesheets (Poole + Lanyon + Syntax) |
| `public/js/` | JavaScript |
| `images/` | Logos, speaker and organizer photos |
